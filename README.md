# Youtube--Clone
This is a YouTube app. In this app, you can Search and see the search results and you can also play the video as per your search results

## Tech Stack
 HTML | CSS | JavaScript

## Deployment
https://www.netlify.com/

## Features

 - search video
 
 - Filters
 
 - Play Video
 
 - Watch Video
